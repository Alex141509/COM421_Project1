Realistic Research: University, Web-based, and Local Resources

This lesson is for students starting research in their first semester of the TCID, IA, or Ph.D. in Technical Communication programs. It offers strategies for developing and improving research skills, ensuring more efficient research.

Getting Started

Accessing Library Services

If you’re newly admitted and don’t have an IIT Hawk Card, you’ll need one for certain library services. Even without a card, you can still access online journals and use the library's search functions.

Using myIIT to Access the Library Website

Access the library website by first logging into myIIT. From there, click on "Library Resources" to use a proxy server connection that allows you to access most resources without reentering login details.

The Library Services icon leads you to the library search website.

The best place to start research is in a scholarly database like Academic Search Premiere (EBSCO).

Research Timelines

Formulating a Timeline

Consider your project deadline when requesting resources. Use this formula:

Time to obtain resource + Time to read resource + Time to integrate resource < Remaining time before deadline

Being Realistic

Don’t assume you can read an entire scholarly book in one day. Choose resources you can realistically read within your timeline.

Bridgette, a graduate student, sets up a timeline to ensure she has enough time to wait for resources, read them, and integrate them into her paper. She uses Google Calendar to track her research goals.

Requesting Resources

Requesting Articles

If articles aren't available for PDF download, you can request them via MyILL. Setup a MyILL account, and once ready, you’ll be notified by email when articles are available for download.

Requesting Books

If you can't find a book in the IIT library system, check I-Share and WorldCat. Keep a list of resources using tools like Google Docs for easy reference.

Case Study: Searching and Requesting Resources

Bridgette makes a note to find the book "English for Engineers" by S.A. Harbarger in the UIC library, as noted in her Google Docs resource list.

Local Libraries

Use local libraries when:

  1. You don’t have enough time for interlibrary loan
  2. The book isn’t available through interlibrary loan

Libraries like UIC’s Daley Library are great for accessing extensive resources. However, always check policies and hours before visiting.

Internet Resources

Some Resources to Consider

Consider using public domain writings, Creative Commons-licensed works, or media files available online to save time.

Using Web Resources for Research

Use resources like Google Scholar, Google Books, or Web of Science to search for additional articles or references.
