How to Organize and Analyze Research Data

For this lesson, you'll need the following:

  • Microsoft Excel
  • PASW (formerly SPSS) statistics software, or free trial download

You should already be familiar with:

  • The basic functions and tools of Excel
  • Descriptive statistics and inferential statistics
  • Research design and how to identify independent variables and dependent variables

This short lesson is intended for graduate and advanced undergraduate students undertaking original research projects. After completing the 30-minute lesson, you will be able to:

  • Format raw research data in an Excel spreadsheet for efficient analysis
  • Run basic descriptive statistics in Excel
  • Prepare the document for statistical analysis in the software package PASW

    • Format Data for Analysis

    Ensure Microsoft Excel is installed on your computer. Open the document ToolBox_DataLesson.xls in Excel and follow along with the lesson.

    In the Raw_Dataset spreadsheet, the name of each variable is entered in the first row of each column.

    1. Each variable name must be unique.
    2. The first variable (in column A) is a unique identifier.
    3. Variable names must start with a letter, so change 4tutorial_types to tutorial_types.
    4. Rename variables intuitively: change use and useful to perc_ease_of_use and perc_usefulness, respectively.
    5. Your spreadsheet should now resemble Figure 1, showing clearly that Column F refers to Perceived Usefulness and Column G refers to Perceived Ease of Use.

    Tutorial type, gender, experience, and task categorical values have been formatted as dichotomous variables.

    Data values (rows) have been entered for each subject in the experiment.

    1. Decide on input conventions and apply them consistently. For example, change "female" in the gender column to "F".
    2. Separate complex data into components. Add columns task1, task2, and task3, and reformat tasks_completed values (Y/Y/N) accordingly.
    3. Double-check for errors, then delete the original tasks_completed column.
    4. Your spreadsheet should now resemble Figure 2.

    Values for each variable are entered in a consistent format.

    Replace categorical data values with "0" or "1" (0=no, 1=yes) for maximum flexibility in testing relationships or correlations.

    Dichotomous variables make it easier to isolate characteristics for analysis. For example, separating experience levels allows testing whether subjects with low experience have different ease of use scores compared to medium and high experience levels.

    1. Copy the Raw_Dataset values into Sheet2 and rename the sheet to Formatted_Data.
    2. In the Formatted_Data sheet, add columns for each distinct categorical value (e.g., tutorial_types, gender, exp_level, task1, task2, task3).
    3. Rename columns and paste values from the original data. Use "0" or "1" for dichotomous data values (0=no, 1=yes) using Edit > Replace.
    4. Your spreadsheet should now resemble Figure 3.
